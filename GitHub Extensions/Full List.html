Curated List of GitHub-related Chrome Extensions

GitHub Code Folding 0.1.1 Enable code folding when viewing files in GitHub. GitHub Comment Editor 0.0.8 Add editor buttons to every text comment area on GitHub. GitHub Diff Navigator 0.1.3 GitHub Diff Navigator allows you to easily navigate through the changes in a file that has been edited on GitHub. GitHub Diff Whitespace Ignore Button 0.1.0 Adds an 'Ignore whitespace' button on any GitHub file diff page. GitHub Diffs 1.0.3 Collapse/expand diff files for easier reviews GitHub Expand Diff 0.0.1.6 A user script for Chrome that allows you to expand the missing lines in a GitHub diff. GitHub Go To Definition 0.1.1 Allows you to jump to the definition of methods, classes, variables, etc. while viewing a diff or file in GitHub GitHub go to line number 1.2.0 Open-source extension which adds a go to line number modal right into GitHub GitHub Highlight Selected 2.1 Highlight selected word in GitHub source view like Sublime Text GitHub Hovercard 1.4.4 Neat hovercards for GitHub. GitHub improved 2.0.6 Adds in-page changeset unfolding, and some other niceties for GitHub's Commit History view. GitHub Repository Size 0.4.1 Automatically adds repository size to GitHub's repository summary github syntax 0.3.0 Adds syntax highlighting to GitHub's split diff pull request view. GitHub with a cape 1.4.4 Open source Chrome Extension which adds a bunch of new features to GitHub. Customize by picking the ones you like! GitHub-Diff-Whitespace 1.3.2 Adds a button to GitHub diff views which toggles whitespace (a la `git diff --ignore-all-space`) github.expandinizr 1.8.0 Chrome extension that improves the GitHub experience Octo Mate 0.16.5 Missing mate of GitHub, making single file download effortless and with more features Octo Preview 1.0.2 Live preview for markdown comments in PRs/Issues on GitHub OctoEdit 0.5.1 Syntax Highlighting mini-editor for GitHub.com Octotree 2.4.3 Code tree for GitHub Pipe it Down 0.1.0 Reduce diff noise in GitHub pull requests Pretty Pull Requests (GitHub) 2.11.1 This extension applies various tweaks to the GitHub pull-request code review pages. Render Whitespace on GitHub 1.3.9 Renders spaces as · and tabs as → in all the code on GitHub. Show Invisibles on GitHub 1.1.3 Spaces or tabs? Keep your coding style consistent by making whitespace characters visible in all GitHub repositories.
